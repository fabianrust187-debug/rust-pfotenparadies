# Pfotenparadies Rust

Website fuer einen mobilen Tierfriseur.

Technik: React, TypeScript, Vite und Tailwind CSS.

Start lokal:

npm install
npm run dev

Build:

npm run build

Vor der Veroeffentlichung bitte Impressum, Datenschutz, echte Telefonnummer, echte E-Mail-Adresse und finale Texte ergaenzen.
